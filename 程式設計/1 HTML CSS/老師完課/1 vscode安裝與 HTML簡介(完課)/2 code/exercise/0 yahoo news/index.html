娛樂報報／王偉忠揪高怡平悠閒吃法餐 華泰二代陳昶福桌邊暢聊

[周刊王CTWANT] 「電視教父」 王偉忠近期忙著宣傳自己監製的舞台劇《謝謝大家收看》，時報周刊CTWANT接獲讀者提供，本月初他與金鐘獎最佳主持人高怡平在台北忠孝東路的法式餐廳用餐，同行的還有另兩名女性友人。用餐期間，餐廳老闆、華泰大飯店集團二代陳昶福都特地出來打招呼並介紹食物，高怡平甚至吃到拍手鼓掌，遇到美食真情流露。

本刊讀者於9月1日晚間在台北市大安區一間法式餐廳用餐時，鄰桌就坐著王偉忠、高怡平與兩名友人，王偉忠穿著長袖白襯衫、短褲，腳踩球鞋一身輕便，邊吃法餐邊喝紅酒，非常享受。席間，頂著招牌光頭加上濃密鬍子的餐廳老闆陳昶福圍著圍裙走到桌邊和王偉忠聊天，王偉忠則將一隻手隨意的放在椅背，愜意悠閒。
