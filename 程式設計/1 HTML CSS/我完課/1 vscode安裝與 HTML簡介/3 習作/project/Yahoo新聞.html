中秋烤肉5大高鹽食物陷阱！
吐司、零卡果凍竟上榜

生活中心／REGINA報導

中秋節即將到來，不少民眾已經開始張羅聚會烤肉的食材與用具，每逢聚會不免大吃大喝，就有醫師出面提醒烤肉雖然是派對之中不可缺少的要角，但一定不能忘記烤肉屬於「高鹽食物」，專家曝光烤肉派對上的「5類高鹽陷阱」，並提醒可能造成的影響，同時也喊出：「記得控制一下鹽分攝取，保護自己的健康哦！」。

維尚泌尿科診所發文提醒民眾在中秋節期間，一定要注意不能疏忽烤肉派對的鹽分攝取情況，專家舉例：「就連你用來夾肉的白吐司，也不例外哦！」，解釋雖然白吐司看似清淡無味，但在製作的過程中，常會加入鹽和奶油增添風味，且吐司通常被當作主食搭配各種肉類和其他食材享用，相較於其他澱粉食品，除了本身就含有鹽分和油脂外，還會吸收更多潛在的鹽份，緊接著專家也曝光烤肉派對上的「5類高鹽陷阱」 ：

  1. 主食類：白吐司、壽麵線、油麵、雞絲麵。
  2. 甜點類：零卡果凍、洋芋片、魷魚絲等。
  3. 飲料類：運動飲料、蔬果汁、楊桃汁、巧克力飲品。
  4. 調味料類：沙茶醬、烤肉醬、辣椒醬、蝦醬、雞湯粉。
  5. 加工食品：香腸、培根、火腿、泡麵。
